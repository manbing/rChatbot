chatbot implementing in Rust

Abstract
----------
Command-line interface, one to one, the simplest chatbot.
Focusing on how to implement chatbot with Rust and Huggingface.

How to use
----------
```
$ cargo run -- --which nemo-instruct-2407 --sample-len 150 --cpu
```
